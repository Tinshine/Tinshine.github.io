strcmp()—字符串的比较

函数原型：

  strcmp()的工作原理为：从两个字符串的首个字符开始，逐位比对其ASCII码值。若前者大于后者则返回1并结束比较，否则返回-1并结束比较；若二者相等，则继续比较下一个。若二者长度相等且两个字符串所有对位都相同，则返回0（如：”abc”和”abc”）。若二者长度不相等，但是前边长度相等的部分字符也相等，且前者较长则返回1（如：”abcd”和”abc”）；否则返回-1（如：”abc”和”abcd”）。
