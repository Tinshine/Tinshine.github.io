直接插入排序算法

发表于 2019-02-27 | 分类于 C++进阶

算法描述

 假设待排数据存放在一维数组L[n]中，待排表在某一时刻的状态可分为三部分：

 第一部分。前面i个数据L[0],···,L[i-1]是已经使用直接插入排序排好的有序子表，简记为A。

 第二部分。第i个元素L[i]是本轮排序待排元素，简记为B。

 第三部分。后面n-i-1个元素L[i+1],···,L[n]是未排序的原始数据，简记为C。

 每一轮排序需要做三件事：

 （1）找到B在A中的插入位置k；

 （2）将L[n]中位置k后所有元素往后移一位；

 （3）将B复制覆盖元素L[k]。

 从第二个元素到最后一个元素，需要进行n-1轮上述排序操作。
