发表于 2019-02-28 | 分类于 C++进阶

算法描述

 快速排序由冒泡排序演进而来，基于分治的算法思想。每次排序时选择一个元素作为“基准值”，经过一轮排序，通过某种划分算法将待排表分为左右两部分，其中左边的元素全都小于基准值，右边元素全都大于基准值。这样，基准值就被放在了它的最终位置。随后，再对左右两个子表递归进行上述操作，最终能够实现排序目的。
